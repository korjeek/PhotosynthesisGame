## State

В данной задаче состояние игры представлено классом `GameState`.

Состояние полностью описывает текущую ситуацию на игровом поле и включает:

- текущий день (`day`)
- количество ресурсов (sun points) у игроков (`mySun`, `opponentSun`)
- текущий счёт (`myScore`, `opponentScore`)
- значение питательных веществ (`nutrients`)
- список всех деревьев (`trees`)
- описание игрового поля (`board`)
- список доступных действий (`possibleActions`)
- состояние оппонента (`opponentIsWaiting`)

Таким образом, `GameState` является полной моделью игрового мира, необходимой для принятия решений и проведения симуляций.

---

## Baseline

В решении используется жадный алгоритм

На каждом ходу:
1. Перебираются все возможные действия.
2. Для каждого действия моделируется развитие игры на несколько шагов вперёд.
3. Полученное состояние оценивается с помощью эвристической функции (`Estimator`).
4. Выбирается действие с максимальной оценкой.

Таким образом, алгоритм принимает решение, основываясь на локально оптимальном выборе с учётом ограниченного прогноза.

---

## Simulation
Для оценки действий используется ограниченная симуляция (forward simulation).

Алгоритм симуляции:
1. Клонируется текущее состояние игры.
2. Применяется выбранное действие.
3. Далее выполняется симуляция на фиксированную глубину:
   - генерируются возможные действия
   - выбирается лучшее действие по жадному принципу
   - применяется к состоянию
4. При действии `WAIT` происходит переход к следующему дню:
   - увеличивается номер дня
   - начисляются солнечные очки с учётом теней
   - снимается состояние `dormant` у деревьев
5. После завершения симуляции итоговое состояние оценивается

Таким образом, симуляция позволяет приблизительно оценить долгосрочные последствия каждого действия.

---

## Estimator
Для оценки состояния используется эвристическая функция, которая учитывает:

- текущий счёт (имеет наибольший вес)
- количество солнечных очков (ресурс)
- количество и размер деревьев
- плодородие клеток
- штраф за избыточное количество семян
- стадию игры (в конце усиливается приоритет завершения деревьев)
