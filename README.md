# 1. State
* InitialState - пустой хексагон состоящий из 37 клеток в виде хексагона. List<Cell> board. Где Cell - это индекс, жирность :D и список индексов его соседей (6 штук).
* State:
  - day - Игра длится 24 дня: 0-23
  - nutrients - базовый балл, который получаю за следующее ВЫПОЛНЕННОЕ действие
  - mySun - мой очки солнца
  - myScore - мой итоговый счет
  - opponentSun - очки солнца противника
  - opponentScore - итговый счет противника
  - opponentIsWaiting - спит ли противник до следующего дня
  - trees - все деревья на поле List<Tree>. Где Tree - это индекс на поле, флаг мой/противника, флаг в активно/нет, размер дерева 0-3.
  - possibleMoves - возможные действия: WAIT, SEED, GROW, COMPLETE
# 2. Baseline
Baseline — это «жадный» алгоритм, который принимает решения здесь и сейчас, без просчета будущего. Он использует список легальных ходов possibleActions, присылаемый сервером, что полностью исключает отправку ошибочных команд (например, попытку вырастить спящее дерево).
Приоритеты действий:
  - COMPLETE: Завершить жизненный цикл дерева, если оно стоит на самой богатой почве (максимальный бонус).
  - GROW: Вырастить дерево. Приоритет отдается деревьям большего размера (превращение 2 в 3 важнее, чем 0 в 1), так как они приносят больше солнца.
  - SEED: Бросить семечко на самую богатую из доступных клеток (с ограничением по количеству семян, чтобы не потратить всё солнце в начале).
  - WAIT: Если нет полезных действий или не хватает солнца.
